# Daily AI News Assistant

An AI-powered Microsoft Power Automate workflow that collects AI news from multiple RSS sources, filters relevant articles using AI, summarizes them, and sends a daily newsletter email.

## Features

- Multi-source RSS news collection
- AI article classification
- HTTP article retrieval with fallback
- AI-generated summaries
- HTML newsletter email design
- Universal article processing pipeline

## Tools Used

- Microsoft Power Automate
- AI Builder
- RSS Connector
- HTTP Connector
- HTML/CSS
- GitHub

## Workflow

RSS Sources → ArticleList Array → AI Classifier → HTTP Retrieval → AI Summary → HTML Newsletter → Email
